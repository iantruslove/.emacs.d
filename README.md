# Emacs dotfiles

## Prerequisites

* Install Cask and Pallet (full instructions at https://github.com/rdallasgray/pallet)
  * `curl -fsSkL https://raw.github.com/cask/cask/master/go | python`
  * cd to ~/.emacs.d and run `~/.cask/bin/cask install`
